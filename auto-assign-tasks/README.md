# Auto Assign Tasks

**Version:** 1.0.0
**Last Updated:** 2026-02-04

## Problem

When creating Task Workflows, you need to assign each task to a specific person which creates manual work reassigning them when you schedule an Event.

## Automator Solution
By using Account Owner, it will redistribute certain tasks (based on Task Type) to that Account owner. It also redistributes tasks based on Task Type for the Booking Owner, Instructor & Administrator.
